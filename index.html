Black Jack Education App

This project dealt with creating our very own education application, where me and my team were to devolop an application on QT to teach other people about a certain subject. For our project we decided to teach black jack, and our app comes with a bunch of features. This gave me more practice with team work and it also was very interesting to be able to create an application as a team without getting the specifics of what we needed to do.

Sprite Editor

With this project, me and a team of 6 made a sprite editor with c++ on QT. In this project we learned how to work together to create an idea for how to implement a sprite editor. This project can save files, load files, it can animate the sprite and you can change different accessability features. This project was a fun way to get to work with others as it was one of the first where I worked with a group of more than 2 people.

Phrase Generator

This project was on of my favorites, and with this project I worked with a partner where we were to develop a phrase generator given a text file full of phrases. This phrase generator in particular created a poem from scratch using the text file. This was a fun project and I love the phrases it generates.
